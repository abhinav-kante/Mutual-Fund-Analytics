# Mutual Fund Analytics

## Project Overview

This project analyzes mutual fund performance using NAV history data obtained through the AMFI API.

## Technologies Used

- Python
- Pandas
- NumPy
- Requests
- SQLAlchemy
- Matplotlib
- Seaborn
- Plotly

## Day 1 Deliverables

- Project Setup
- Data Ingestion
- Live NAV Fetching
- Data Quality Analysis

## Schemes Analyzed

- SBI Bluechip
- ICICI Bluechip
- Nippon Large Cap
- Axis Bluechip
- Kotak Bluechip
